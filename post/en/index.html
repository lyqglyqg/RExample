Elastic Net

For this, we use the functions and parameters from the glmnet package. In the glmnet package, the alpha value determines the type of model:

Lambda represents the overall degree of regularization. The larger this value, the greater the penalty, resulting in fewer retained variables and lower model complexity. Alpha is the proportion of L1 regularization. When alpha = 1, it is a Lasso model. When alpha = 0, it is a Ridge model. When 1 > alpha > 0, it is an Elastic Net model.

First, set alpha to 0.5. Actually, we need to find the optimal alpha and lambda values. We can try alpha values between 0 and 1 to find the best values.

set.seed(123)
fit <- glmnet(train.x, train.y, family = "cox", alpha = 0.5, nfolds = 10)
plot(fit, label = T, xvar = "lambda")

Cross Validation

fit_cv <- cv.glmnet(train.x, train.y, family = "cox", alpha = 0.5, nfolds = 10)
plot(fit_cv)

Optimal model

Stratified Cox regression

To perform stratified Cox regression with regularization using the glmnet package, you need to use the stratifySurv function to create stratified survival objects, since glmnet does not support formula interfaces directly. Here’s how you can implement it:

  • Stratify the survival data.
  • Perform the Cox regression with regularization.

Here’s an example using the lung cancer data. We stratify the data into 5 strata.

strata <- rep(1:5, length.out = 117)
y2 <- stratifySurv(train.y, strata)  # stratify y
str(y2[1:6])
##  'stratifySurv' num [1:6, 1:2] 361 218 166 170 567 613
##  - attr(*, "dimnames")=List of 2
##   ..$ : NULL
##   ..$ : chr [1:2] "time" "status"
##  - attr(*, "type")= chr "right"
##  - attr(*, "strata")= int [1:6] 1 2 3 4 5 1

fit_strata <- glmnet(train.x, y2, family = "cox", alpha = 0.5)
plot(fit_strata, label = T, xvar = "lambda")


cv.fit_strata <- cv.glmnet(train.x, y2, family = "cox", nfolds = 10)
plot(cv.fit_strata)


FinalModel <- glmnet(train.x, y2, family = "cox", alpha = 0.5, lambda = cv.fit_strata$lambda.min)
apply(predict(FinalModel, train.x), 2, Cindex, y = train.y)
